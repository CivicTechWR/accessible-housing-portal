# Context

## Domain Vocabulary

### Listing

An affordable housing opportunity shown to renters and managed by a listing author. A Listing combines property details, unit details, application details, images, and custom field answers.

### Listing Intake

The create, draft, autosave, publish, and update flow that turns submitted Listing form data into the persisted Listing graph.

### Listing Projection

The read shape of a Listing for a specific caller, such as search summaries, details pages, owner listing rows, or editor data.

### Listing Query

The canonical URL-driven filter and sort state used to fetch Listings from both server-rendered pages and browser-driven refinements.

### Custom Listing Field

An administrator-defined field that can add searchable, public, required, or internal metadata to Listings.

### Custom Listing Field Category

The canonical grouping label for Custom Listing Fields. Categories are stored normalized and displayed as readable labels.

### Custom Listing Fields Dashboard Workflow

The administrator interaction model for searching, selecting, editing, bulk editing, deleting, and reordering Custom Listing Fields.
